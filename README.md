# ⌘ Prompt Dev Tracker

Un outil personnel pour sauvegarder, organiser et retrouver rapidement les prompts IA qui fonctionnent vraiment en développement web.

> Construit en Vanilla JS, sans framework, sans dépendances. Fonctionne directement dans le navigateur.
>
> ---
>
> ## Pourquoi cet outil ?
>
> Quand on code avec l'IA (Claude, ChatGPT, Copilot...), on réutilise souvent les mêmes prompts efficaces — mais on les oublie ou on les perd. Ce tracker permet de les centraliser, les noter et les copier en un clic.
>
> ---
>
> ## Fonctionnalités
>
> - ➕ Ajouter / modifier / supprimer des prompts
> - - 🏷️ Tags par techno : JS, CSS, Git, API, Debug, Architecture...
>   - - ⭐ Notation de 1 à 5 étoiles
>     - - ✓ Résultat : Efficace / Partiel / Inefficace
>       - - 🔍 Recherche full-text sur titre, contenu et tags
>         - - 📂 Filtres dans la sidebar (par tag, top rated, efficaces)
>           - - ⎘ Copier en un clic pour coller directement dans son outil IA
>             - - 💾 Persistance locale via localStorage
>               - - ↑ Export JSON pour sauvegarder ou migrer ses données
>                 - - 📦 6 prompts d'exemple préchargés (debug JS, review de code, Git, API REST...)
>
